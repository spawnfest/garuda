# Garuda
> ### A multiplayer game server framework for BEAM
  > Build and run game servers intuitively.
  
Garuda  is an Authoritative Multiplayer Game Server for phoenix framework.

The project focuses on providing a *game framework system*, *matchmaking*, *live game-session monitoring system* and ease of usage both on server-side and client-side, by leveraging the powerful phoenix framework.   

The goal of the framework is to be a standard netcode & matchmaking solution for all type of games.

Current feature list. 
-   WebSocket-based communication (Will support more transports layer in future, thanks to phoenix)
-   Simple API in the server-side and client-side.
-   Matchmaking clients into game sessions.
-   Realtime interactive game session monitoring.
